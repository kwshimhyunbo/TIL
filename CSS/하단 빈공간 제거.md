# 하단 빈공간 제거

퍼블리싱을 할 때 하단의 빈공간은 다음과 같이 제거할 수 있다.

                html,
                body {
                	margin:0;
                	padding:0;
                	height:100%;
                }
                #wrapper {
                	min-height:100%;   
                	position:relative;
                }
                #header {
                	background:#ededed;
                	padding:10px;
                }
                #content {
                	padding-bottom:100px;
                }
                #footer {
                	background:#ffab62;
                	width:100%;
                	height:100px;
                	position:absolute;
                	bottom:0;
                	left:0;
                }
