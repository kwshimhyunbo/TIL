# 그리드의 필요성

flex는 충분히 좋은 시스템이지만 아이템 사이의 간격을 표현하는것이 완벽하지는 않았다.

예를들어, space arround 라는 효과를 줄때 다음과 같이 표현되어 상식적이지 못한 방식으로 표현해버린다.

이를 개선하여 6 옆에 7을 바로 놓을 수 있도록 그리드가 표현해줄 수 있다.

#### FLEX
```
1 2 3 4 5
6       7
```

#### GRID
```
1 2 3 4 5
6 7
```
